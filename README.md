### 如何将 HTML 文件转换为 Figma 文件

-   1. 部署当前站点，拿到网站的 URL
-   2. 在 Figma 中新建一个文件
-   3. 打开 Figma 的插件 Store，搜索 `HTML to Figma`
-   4. 安装插件，然后在插件中输入网站的 URL 即可将网页转换为 Figma 文件
